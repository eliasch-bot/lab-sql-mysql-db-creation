## Introducción

Bienvenido a este laboratorio donde practicarás cómo diseñar, crear y gestionar una base de datos. En este ejercicio, asumirás el rol de experto en datos en una empresa de concesionarios de coches que vende vehículos nuevos de varias marcas y modelos. Tu tarea será diseñar, crear y administrar una base de datos que registre información sobre coches, vendedores, clientes y facturas. 

El laboratorio consta de tres desafíos que te ayudarán a lograr esta tarea:

1. Diseñar la base de datos identificando las entidades y sus relaciones.
2. Crear la base de datos y las tablas en MySQL utilizando las sentencias `CREATE DATABASE` y `CREATE TABLE`.
3. Insertar datos en las tablas utilizando la sentencia `INSERT INTO`.

En el desafío adicional podrás actualizar registros con `UPDATE` y eliminarlos con `DELETE`.

Al finalizar, habrás adquirido experiencia práctica en el diseño, creación y gestión de bases de datos relacionales. ¡Comencemos!
